# Database : 
1 Create a database
2 Name it `cctc-guide`
3 Export the database `cctc-guided.sql` in `db`  folder

## Table: `teacher`

### Structure

- `user_id` (int): Primary key for teacher users.
- `email` (varchar): Teacher's email.
- `password` (varchar): Teacher's hashed password.
